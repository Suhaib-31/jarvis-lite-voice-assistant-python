# Jarvis Lite Voice Assistant 🎙️

## Overview

Jarvis Lite is a Python-based voice assistant that can listen to voice commands, speak responses, open websites, and provide useful information.

---

## Features

- Voice Input
- Voice Output
- Open Google
- Open YouTube
- Tell Current Time
- Exit Command

---

## Technologies Used

- Python 3
- SpeechRecognition
- pyttsx3
- PyAudio

---

## Installation

```bash
pip install SpeechRecognition
pip install pyttsx3
pip install pyaudio
```

---

## Run

```bash
python main.py
```

---

## Commands

- Open Google
- Open YouTube
- What is the time?
